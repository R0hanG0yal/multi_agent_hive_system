Frequently Asked Questions
What is the Hive Multi-Agent System?
Hive is a production-grade AI assistant built on a multi-agent architecture. Instead of a single LLM processing everything, your query is routed to specialized expert nodes (Food, Business, Coding, Research) which work in parallel. Their outputs are merged by a Q-Learning-powered Decision Engine that continuously learns which agent performs best for each type of question.
How do I create an account?
Click "Sign Up" on the login page and enter your email and password. Your password is immediately hashed using bcrypt before being stored in our SQLite database — we never store plain-text passwords. After registration you'll be redirected to log in and instantly access the chat.
Why does the AI sometimes route to multiple agents?
The Router Planner computes a relevance score for each specialist node based on your query. If two or more nodes score above the activation threshold (0.15), they both process your question in parallel. The Decision Engine then compares their confidence scores (weighted by Q-values) and either picks the best or uses the LLM to merge them into a superior answer.
Does Hive have real-time internet access?
Yes! For queries containing keywords like "weather", "today", "news", "current", or "latest", Hive automatically triggers a DuckDuckGo web search, retrieves the top 3 results, and injects them directly into the ResearchNode's context before generating a response. This is why MAHA can tell you today's weather accurately.
What do 👍 and 👎 actually do?
They directly train the Hive's Q-Learning controller in real-time. A 👍 sends a reward signal of +1.0 to the Bellman equation, increasing the Q-value of the agent that answered you. A 👎 sends -0.5, decreasing it. Over time this updates which agent scores highest for each domain, making Hive progressively smarter for your specific usage patterns.
Is my chat history stored permanently?
Chat sessions are persisted in your browser's localStorage, so they survive page refreshes and tab closures. The Hive sidebar automatically restores all your previous conversations. Long-term vector memories (facts about you the AI learns) are stored in RAM for the current server session — SQLite persistence for memory is on the roadmap.
What do Easy, Medium, Hard task labels mean?
Every interaction is classified by token consumption. ≤300 tokens = Easy (quick factual questions), 301–700 = Medium (multi-step reasoning), >700 = Hard (complex code generation, long-form synthesis). Future updates will include a live breakdown of these stats.
What is epsilon (ε) in the dashboard?
Epsilon controls the Exploration vs. Exploitation trade-off in Q-Learning. At ε=0.3 (start), there's a 30% chance the Hive randomly tests different agents instead of always picking the best-known one — this helps it discover which agents work well for new query types. Over thousands of interactions, ε decays toward 0.05 (mostly exploitation).
